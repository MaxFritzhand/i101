Name and define the programming terms we learned today.

Function

  • a type of procedure or routine that carries out a task
  • EX: print is a function that displays hello world

Argument

  • a value that is passed to a function.
  • EX: "hello world" is an argument. It's what goes inside the ()

String

  • a sequence of characters
  • EX: "this is string, because its surrounded by quoatation"

Variable

  • a container, given a name, that represents a value

Integer

  • a whole number (a number without a decimal point)
  • EX: 1,-1,0,500,124954,-10000

Why is it important to keep track of data types?

  • You will eventually have dozens of variables you have to keep track of
  • Plus there are different types of data, including strings, integers, lists, dictionaries, floats, Booleams, etc.
