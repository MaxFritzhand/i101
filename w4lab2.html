Week 4 Lab 2 -- Python

What does the len() function do?

  • The len() function allows us to easily compute the length of a string.
  • In other words, this function returns the number of characters in a string.

What does "hello"[3] equal?

  • If print was in front of "hello"[3], it would print l.
  • Since no print is in front of it, nothing would execute.
